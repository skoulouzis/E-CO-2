ECO2Controller Resource

The EDISON COmpetencies ClassificatiOn (E-CO-2) service is an automated tool designed to support gap analysis. It can identify the similarity of a document against a set of predefined categories. It can therefore be used to perform a gap analysis based on the EDISON DS-taxonomy to identify mismatches between education and industry. Moreover, students, practitioners educators and other stake holders can use these tools to identify the gaps in their skills and competences.
